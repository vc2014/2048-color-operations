Color Operations:

Blue = Addition (+)
Red = Subtraction (-)
Green = Multiplication (×)
Orange = Division (÷)
Gray = Neutral

How to Play:

Use arrow keys to move tiles. When two tiles with the same number touch, they merge using the operation determined by their colors!

Try to reach the target number by strategically combining tiles with different operations.
